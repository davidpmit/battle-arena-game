# 🎮 זירת הקרב - משחק אסטרטגיה

משחק אסטרטגי בזמן אמת עם מכניקות קרב מרגשות!

## 🎯 איך לשחק

### מטרת המשחק
השמד את המגדל הראשי של האויב לפני שהוא משמיד את שלך!

### בקרות
1. **בחירת כרטיסים**: לחץ על כרטיס יחידה בחלק התחתון
2. **הנחת יחידות**: לחץ על המפה בחצי התחתון כדי להניח את היחידה
3. **אליקסיר**: כל יחידה עולה אליקסיר - המתין שהאליקסיר יתמלא

### סוגי יחידות

#### ⚔️ לוחם (2 אליקסיר)
- בריאות: בינונית
- נזק: בינוני
- טווח: קצר
- מהירות: מהירה
- מתאים ל: התקפות מהירות

#### 🏹 קשת (3 אליקסיר)
- בריאות: נמוכה
- נזק: גבוה
- טווח: ארוך
- מהירות: בינונית
- מתאים ל: תקיפה מרחוק

#### 🛡️ שריון (4 אליקסיר)
- בריאות: גבוהה מאוד
- נזק: נמוך
- טווח: קצר
- מהירות: איטית
- מתאים ל: הגנה וספיגת נזק

#### 🔮 קוסם (5 אליקסיר)
- בריאות: בינונית-נמוכה
- נזק: גבוה מאוד
- טווח: ארוך מאוד
- מהירות: איטית
- מתאים ל: נזק אזורי עוצמתי

### אסטרטגיות

1. **איזון**: שמור על איזון בין התקפה להגנה
2. **ניהול אליקסיר**: אל תשתמש בכל האליקסיר בבת אחת
3. **שילוב יחידות**: שלב יחידות שונות ליתרון טקטי
4. **הגנת המגדלים**: הגן על המגדלים הצדדיים כדי להגן על הראשי

## 🚀 הפעלת המשחק

### דרך 1: שרת Python
```bash
cd battle-arena-game
python3 -m http.server 8000
```
פתח דפדפן וגש ל: `http://localhost:8000`

### דרך 2: פתיחה ישירה
פשוט פתח את הקובץ `index.html` בדפדפן
